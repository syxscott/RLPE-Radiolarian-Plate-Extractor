## ❓ 常见问题

**Q: 哪些浏览器支持？**
A: Chrome, Firefox, Safari, Edge (现代版本都支持)

**Q: 在旧浏览器上会怎样？**
A: 基本功能保持，但没有渐变、模糊等高级效果

**Q: 可以自定义颜色吗？**
A: 可以，编辑 style.css 中的 CSS 变量

**Q: 响应式在哪里配置？**
A: 在 style.css 的 @media 查询中

**Q: 可以添加暗黑模式吗？**
A: 可以，添加额外的 CSS 变量和媒体查询
