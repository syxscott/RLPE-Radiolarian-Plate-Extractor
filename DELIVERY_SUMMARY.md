## 🚀 部署和使用

### 系统要求
- Python 3.11+
- FastAPI 和 Uvicorn (已列在 requirements.txt)
- 现代浏览器 (Chrome/Firefox/Safari/Edge)
- GROBID 服务 (http://localhost:8070)

### 部署步骤 (仅需 2 步！)

```bash
# 步骤 1: 安装依赖
pip install fastapi uvicorn pydantic

# 步骤 2: 启动服务
python run_web_server.py

# 完成！打开浏览器访问
http://localhost:8000
```

### 首次使用 (仅需 3 步！)

```
1. 拖拽上传 PDF
2. 点击"开始处理"
3. 查看结果
```
