# RLPE Web 界面 - 快速参考卡片

## 🚀 30 秒快速开始

### Step 1️⃣ : 启动服务器

**Windows**:
```
python run_web_server.py
```

**Linux/macOS**:
```
python3 run_web_server.py
```

### Step 2️⃣ : 打开浏览器

```
http://localhost:8000
```

### Step 3️⃣ : 开始使用！

## ❓ 快速问题解答

### Q: 网页打不开？
**A**: 
1. 确认服务器运行 (看终端输出)
2. 尝试 `http://localhost:8000`
3. 清除浏览器缓存 (Ctrl+Shift+Delete)

### Q: "无法连接" 警告？
**A**: 
检查 GROBID 运行: `http://localhost:8070`

### Q: 处理需要多久？
**A**: 
- 小 PDF (5MB): 几秒钟
- 中 PDF (50MB): 几分钟
- 大 PDF (100MB+): 10+ 分钟

### Q: 结果准确度如何？
**A**: 
- 高置信度 (80%+): 通常准确
- 中等置信度 (50-80%): 需要审查
- 低置信度 (<50%): 使用纠正功能

### Q: 能否同时处理多个 PDF？
**A**: 
是的！上传后都会排队处理。

### Q: 纠正记录保存在哪？
**A**: 
`service_work/corrections/corrections.jsonl`

## 💡 使用技巧

### 技巧 1: 默认值就很好
```
大多数情况下，默认配置已经很好。
新手不需要调整参数。
```

### 技巧 2: 小规模测试先行
```
上传 10 个 PDF 进行测试
查看结果质量
再决定是否全量处理
```

### 技巧 3: 利用搜索功能
```
结果表格中输入论文 ID 或物种名
快速找到特定项目
```

### 技巧 4: 定期导出结果
```
处理重要数据后立即导出
防止浏览器清除数据
```

### 技巧 5: 批量纠正
```
一次性纠正多个低置信度项
提高数据准确度
```
